## Privacy and Security

GaiaThinker is built for schools and designed with student
safety as the highest priority.

### Data Privacy
- ✅ No student accounts required for testing
- ✅ Fully compliant with **BC FIPPA**
- ✅ Fully compliant with **PIPEDA**
- ✅ **Canadian data residency** — all data stored in
  Supabase Canada East region
- ✅ Student data **never sold, shared, or used for advertising**
- ✅ No personally identifiable information collected

### Platform Security
- ✅ Cloudflare security infrastructure
- ✅ All connections encrypted via HTTPS (TLS 1.2+)
- ✅ Secure edge functions for all AI interactions
- ✅ API keys never exposed to client-side code
- ✅ Row-level security on all database tables

### Content Safety
- ✅ All data sourced from BC government and peer-reviewed research
- ✅ Age-appropriate for Grades 9-12
- ✅ Solutions-focused — no graphic or anxiety-inducing content
- ✅ Indigenous content developed with respect for
  First Peoples Principles of Learning

### Family Friendly
- ✅ Parents can participate alongside students
- ✅ No social media integration
- ✅ No in-app purchases
- ✅ No advertising. Ever.

## Getting Started (For Developers)

```bash
# Clone the repository
git clone https://github.com/ALi-Fard/gaiathinker.git

# Navigate to project directory
cd gaiathinker

# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Add your keys to .env:
# VITE_SUPABASE_URL=your_supabase_url
# VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
# ANTHROPIC_API_KEY=your_anthropic_api_key

# Start development server
npm run dev

# Open in browser
# http://localhost:5173
```
